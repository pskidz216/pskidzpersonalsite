Paul Skidmore · Bond No. 9 · Popup Strategy · Three Variants

The popup, on Bond's terms.

The standard 3-step popup runs 10% / 15% off discount steps. Bond's price point ($265–$490) and editorial register ("we don't run sale theater") forbid it. Three Bond-specific variants follow — each rebuilt around exclusivity, discovery, or cashback instead of percent-off discount. Each paired to a homepage option from Page 01 so the popup voice agrees with the site voice. North Star: opt-in rate 15%+ email · 5%+ phone · with Revenue Per Session of opted-in >> non-opted-in.

00 / Bond's situation

Why the default Perfect Popup doesn't ship as written.

The canonical 3-step popup uses a 10% / 15% off double discount. That works for a $50 mass-prestige beauty brand. It cheapens a $265–$490 niche-luxury fragrance — and Bond's editorial register explicitly forbids it. Below is the four-factor situation that drives popup variant selection for Bond specifically.

02 / Reference patterns

How niche-luxury houses run their opt-in.

The default Perfect Popup is calibrated for mass-prestige and DTC. Below: how five reference brands at Bond's register handle email/phone capture differently. The pattern: discovery, exclusivity, or service — never percent-off discount.

Le Labo
No popup · footer only
Famously austere. No popup, no exit-intent, no discount. Footer signup with editorial copy. Opt-in is lower volume, but every email is high-intent. Brand-coherent but leaves growth on the table.
Parfums de Marly
Single-step · sample offer
Light delayed popup (10s) offering "Complimentary sample with your first order." No %, no urgency. Single-step email capture. Discovery / sample is the value exchange.
Frederic Malle
Service-led capture
"Speak with a perfumer" CTA in the buy box that triggers a contact form. Captures email + phone for high-intent customers as a service, not a discount.
Byredo
Quiz-led discovery
"Find your scent" quiz reachable from the homepage. Quiz output emails the user their match. Email capture is built into the quiz flow, never separately. Output also segments by note family.
MFK
VIP / private-list angle
Email opt-in framed as "Be the first to know" for limited editions. Exclusivity language replaces discount language. Higher-AOV customers are recognition-seekers, not coupon-seekers.
Bond No. 9 today
Footer only
Single-step footer form. No trigger. No SMS. No value exchange beyond "be the first to hear." Closer to MFK in restraint, but without the editorial copy or VIP framing that earns the lower volume.

05 / Option C

The popup voice has to agree with the page voice. A cashback popup with an editorial homepage reads inconsistent. A 3-step exclusivity popup with a fast-conversion homepage reads inconsistent. The pairings below match register across the entire above-the-fold experience.

08 / Cross-cuts · apply to all three options

Trigger logic, frequency, and brand discipline — identical across all three.

Below is the framework that applies to whichever option ships. Trigger rules, frequency caps, dim-treatment, close behavior, and voice boundaries are all option-independent.

Trigger + frequency

  • — Trigger: Exit-intent OR 50% scroll-depth. Whichever fires first. NEVER load-immediate (doubles bounce rate). Option B may use 10s delay as the third trigger because the quiz is the value, but Options A and C must use exit-intent / scroll only.
  • — Frequency cap: 1 per 7 days per user. Cookie-based. Session count = 1 only (do not show on second session within 24h). If the user closes the popup, do not re-fire on the same session.
  • — New visitors only by default. Attentive display rule: session count = 1 AND not yet subscribed. Returning subscribers get a different Creative (or none).
  • — No popup on cart, checkout, account, or PDPs above the fold. Popup fires on home, collection, neighborhood index, neighborhood detail, story / about pages only.

Brand discipline

  • — Bond voice, every line. Cabinet Grotesk display headlines, Karla body, gold accent on eyebrows. No exclamation marks. No "AMAZING" / "INCREDIBLE" / "Save Big". Editorial register, NYC-native voice.
  • — No urgency, no countdown, no scarcity. Per Bond's editorial register. No "OFFER ENDS IN 4:32" timers. No "ONLY 3 LEFT IN STOCK" badges in the popup. No "LAST CHANCE" copy.
  • — Bond Gold (#C5A55A) for accents only, never for button backgrounds. Per Bond's logo treatment rules. CTAs are black or white; gold is for eyebrows, hairlines, and "claim my $25" highlight underlines only. Option C is the exception (gold CTA on the cashback step) because the gold IS the offer.
  • — Wordmark allowed in the popup but never larger than 90px width. The popup is not the masthead. The script wordmark may appear muted in the corner if needed, but the popup card carries its own typographic hierarchy.

Confirmation state

  • — "Continue browsing" CTA, never "Save for later" or "Got it." Routes vary by option: A → /pages/new-york-neighborhoods · B → matched fragrance PDP · C → /pages/new-york-neighborhoods.
  • — No discount code displayed in confirmation. Options A and B never display a code. Option C displays "$25 locked in" in plain language, not a code.
  • — Cookie set immediately on confirmation. Suppresses re-fire for 7 days. Subscribed customers never see the popup again until segmented otherwise.

Anti-patterns · never ship

✕ Page-load popup Doubles bounce rate. Bond removed this from the previous SMS popup — do not bring it back. Trigger must be exit-intent / scroll / time-delay (≥7s) at minimum.
✕ % off discount in copy Off register. "10% off" / "15% off" / "Save 20%" all forbidden. Substitute: cashback (Option C), exclusivity (Option A), discovery (Option B).
✕ Email + phone on same step Kills opt-in. Option A separates email (Step 2) from phone (Step 3). Option C drops phone entirely. Never on the same form.
✕ Hidden / unclosable X Illegal in some jurisdictions, always bad UX. X close button visible in top-right of every step at minimum 14×14px hit target on desktop, 24×24px on mobile.
✕ Stacked popups in one session One popup per session, period. No newsletter + SMS + country selector + age-gate cascade. If multiple are needed (rare), sequence them across sessions, not stacked.
✕ "Save for Later" final CTA Use "Continue browsing" instead. "Save for Later" reads as the user is leaving the site — exactly the wrong signal at the moment of opt-in.

10 / Pre-launch QA · ship checklist

Run every box before turning a popup on.

All three options share the same QA gate. Skip nothing. The most common popup failures are the boring ones — discount code not auto-applying, X close hidden behind chat bubble on iOS Safari, popup re-firing on every page load because cookie wasn't set. Test on real devices, not BrowserStack alone.

11 / Measuring success

The metrics that matter.

Opt-in rate alone is a vanity metric. The real test is whether opted-in users carry materially higher Revenue Per Session than non-opted-in. If opt-in is 25% but Revenue Per Session of opt-ins is the same as non-opt-ins, the popup is collecting low-quality leads — tighten the trigger.

Primary · email opt-in
15%+
Across all three options. Question-based (B) targets 22–28%. Cashback (C) targets 25–32%.
Primary · phone opt-in
5%+
Option A only. 3–5% range without discount, 5–8% with discount — Bond's exclusivity frame is closer to 4–6%.
Secondary · RPS lift
2×+
Opted-in user RPS should be at minimum 2× non-opted-in. If flat, popup is collecting low-quality leads.
Long-term · LTV / CAC
3:1+
Email + SMS flows triggered post-opt-in must produce LTV at 3× the cost of acquiring the opt-in (popup tool cost + cashback margin if Option C).
